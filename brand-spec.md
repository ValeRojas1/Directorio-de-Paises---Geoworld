- `--bg`:      oklch(100% 0 0)      /* #FFFFFF (Blanco) */

- `--fg`:      oklch(25% 0.04 250)  /* #1A2B3C (Texto Navy) */
- `--muted`:   oklch(55% 0.02 240)  /* #6B7A8D (Gris) */
- `--border`:  oklch(85% 0.06 230)  /* #ACD2F1 (Azul border) */

## Typography
- **Font Family**: 'Inter', sans-serif (Google Fonts)
- **H1**: 48px bold
- **H2**: 32px semibold
- **H3**: 20px semibold
- **Body**: 16px regular
- **Label**: 13px regular

## Layout Posture
- **Style**: Clean, modern, educational, friendly.
- **Radii**: Cards 12px, Buttons 10px, Pills 20px, Inputs 12px.
- **Shadows**: Soft (0 4px 16px rgba(0,0,0,0.08)).
- **Spacing**: Generous (24px, 32px, 48px gaps).
- **Constraints**: No stock photography, flags from flagcdn.com, emoji icons only.
